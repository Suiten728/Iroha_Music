いろはMusicBotはかざま隊の集いの場内で音楽の再生や加工をします。こちらは単サーバーを目的としていますが、他のどのサーバーでも使用できる設定となっています。詳しくは[Suiten](https://github.com/Suiten728)までご連絡下さい。

## 🛠機能
ぽこべぇ管理人Botにはたくさんの機能が備え付けられています。追加予定及び追加済機能は以下のとおりです。</br>
💡実装済みは🔵、未実装又は協議中は🔴で分けています。</br></br>
### 🔵IM!play｜/play
音楽の再生を開始します。VCにすでに入っている必要があります。</br>
### 🔵IM!stop｜/stop
音楽の再生を停止します。自動で退室します。</br>
### 🔵IM!now｜/now
現在再生されている音楽情報を送信します。</br>
### 🔵IM!ping
レイテンシーを計測します。速度によって文字が変わります。</br>
### 🔵IM!eval
コード実行コマンドです。コードブロック(```)にしなくても動作します。サーバーオーナーのみ使用可能です。</br>
